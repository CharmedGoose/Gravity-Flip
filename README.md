# Gravity Flip

[itch.io](https://charmedgoose.itch.io/gravity-flip)  
Flip gravity to avoid asteroids

## Controls

Space or Left Click to flip gravity

## Running Locally

Clone this repository:  
`git clone https://github.com/CharmedGoose/Gravity-Flip.git`

Then run:  
`npm run build`

Then open:  
`dist/index.html`
